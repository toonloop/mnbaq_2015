Installation instructions
=========================

Ubuntu 14.04
------------
* build and install toonloop from the develop branch
* build toonplayer from the develop branch

Install as well::

  sudo apt-get install inotify-tools lunch

ln -s ~/src/mnbaq_2015/launch_toonloop.lunch ./.lunchrc
ln -s ~/src/mnbaq_2015/launch_toonloop.lunch ./.lunchrc
ln -s ~/src/mnbaq_2015/launch_toonplayer.lunch ./.lunchrc
